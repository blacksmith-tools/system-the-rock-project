## Install

```bash
npm install
```

## Usage

```bash 
mongod
```

```bash 
mongoimport --db the-rock --collection foods --file ./resources/data/the-rock/foods.json
mongoimport --db the-rock --collection users --file ./resources/data/the-rock/users.json
```

```bash
node server
```
